Bienvenue dans MoMoMotus

Les règles du jeu

  • Vous devez entrer un mot de 8 lettres commençant par la lettre indiquée sur le plateau.
  • Les lettres bien-placées s'affichent dans un cercle jaune, puis sur la ligne suivante.
  • Les lettres mal-placées s'affichent dans un cercle orange.
  • Vous disposez d'un maximum de 10 tentatives pour découvrir le mot caché.

Ce projet a été réalisé dans un but purement pédagogique.
