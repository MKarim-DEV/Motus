Les règles du jeu

  • Vous devez entrer un mot de 8 lettres commençant par la lettre indiquées sur le plateau
  • Les lettres bien-placées s'affichent dans un cercle jaune, puis sur la ligne suivante
  • Les lettres mal-placées s'affichent dans un cercle orange
  • Les lettres non présentes s'affichent dans un cercle rouge
  • Vous disposez d'un maximum de 10 tentatives pour découvrir le mot cachés

Ce projet a été réalisé dans un but purement pédagogique.
